## Starting 🚀
These instructions will allow you to get a copy of the project running on your local machine for development and testing purposes.

## Installation 🔧
To install and run this proyect just type and execute
```
$ npm install
$ npm run serve
```

## Deployment 📦
```
$ npm run build
```
